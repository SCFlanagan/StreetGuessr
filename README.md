# StreetGuessr

StreetGuessr is a clone of the game [Geoguessr](https://geoguessr.com), replicating its exact design and game function.

### Features:

### Tech:
* JavaScript
* React
* Redux
* Node
* Express
* MongoDB
* Mongoose
* Jasmine
* Google Maps API
* Heroku

Deployed at: https://streetguessr.herokuapp.com
